# vue-counter-html-base

## 要求
- Fork该项目仓库到你的账号下
- 实现下面的功能列表

## 功能列表

- 页面上需要显示两个按钮 ("+" "-")
- 页面上需要显示一个默认值为 0 的数字标签 (span)
- 点击 "+" 号按钮，数字标签应该加一，显示数字1，并以此类推
- 点击 "-" 号按钮，数字标签应该减一，显示数字-1，并以此类推
